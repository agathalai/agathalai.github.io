Help

The SoFi Connect tool works with the Thinkstep SoFi software platform to help streamline your sustainability performance reporting by retrieving and consolidating all of your utility data in one place. Once the data has been loaded to SoFi Connect, the data is processed and business rules are applied before it is sent to the SoFi software platform.

Supported browsers

SoFi Connect currently works best on the following browsers:

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge

How to change your password

After you log into SoFi Connect for the first time, it is highly recommended that you change your password. You can also choose to change your password at any time.

  1. In the top-right corner, click your name.

    The Manage your account screen is displayed. By default, the Password tab is selected.

  2. Enter your current password.
  3. Enter your new password.
  4. Enter your new password again to confirm that it is correct.
  5. Select to complete the change

Next time you login, you can use your new password.

Add utility data to SoFi Connect

You can add utility data to SoFi Connect by selecting any of the following options from the top menu:

  • Data Files - select this option to manually upload data from a .CSV (comma separated value) file from your local computer.

    Select the file you want to add then select

    NOTE: Add a separate .CSV file for each reporting period or SoFi Connect may not process the data correctly. Each file should include one extra calendar month at the beginning and end of the reporting period to ensure that all the relevant data is correctly processed.

    For example, if you are uploading data for December 2017, include data from the November and January utility bill to ensure that all the December dates are included.

  • Inbox - select this option to see the .CSV files that were automatically loaded to your pre-configured S3 bucket, FTP, SFTP, or server.

    Check the box beside any new data files then select to have SoFi Connect process the data.

  • PM Connect > Utility Data or EnergyStar Score - select this option to download different types of data directly from your Energy Star Portfolio Manager account.

    Select the type of data you want and the reporting period then select

  • Supplier - select this option to retrieve data directly from your supplier.

    Select a supplier from this menu, then select the period of data you want and click

Be aware that adding data to SoFi Connect may take several minutes. Be patient and do not close the browser window or tab.

Manage utility data in SoFi Connect

When you select Details beside any record on the Utility Data Files dashboard, you see the Utility Data page for that record. On the Utility Data page, you will see all the account information that have been retrieved from the data source.

On the right is a log of each time the data is sent to the SoFi software.

If you want to know more about each record on the page, just click Details:

  1. After you select Details on the Utility Data page, you see the Accounts page.
    This page lists all the records managed by the selected customer account.
  2. After you select one of the records on the Accounts page, you see the full utility data page for the selected account record.

Add additional supplier accounts

By default, your version of SoFi Connect should be pre-configured with your utility supplier accounts. However, if you need to add additional accounts for any of the suppliers, complete the following steps.

  1. From the top menu bar, select Supplier then select one of the listed supplier options.
  2. In the top-right side of the following screen, select the supplier accounts button.
    The supplier accounts list page is displayed.
  3. In the top-right side of the screen, select .
  4. On the Add account screen, enter the following information:
    • Initials: a short initial or code to help you identify the account.
    • User: the account username.
    • Password: the password for the account.
  5. Select .

The new account is added to SoFi Connect.
