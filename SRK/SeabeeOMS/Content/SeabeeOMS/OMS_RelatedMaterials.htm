Objectives

This section provides key documents related to the EL and TL TMFs, such as site characterization, design, operation, inspections, and preliminary decommissioning and reclamation plans.

Location and Hyperlinks

The material can be found on SGO’s server at OMS for Seabee Tailings (sharepoint.com):

The reference for each document is hyperlinked to the above directory. External reviewers will need to request copies of related material.
