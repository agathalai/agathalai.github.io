The Seabee Mine Environmental Database application has been developed on a standard SQL Server database and is currently hosted within SRK’s GIS cloud server on Amazon Web services. The web interface allows for easy selection of a surveillance station and viewing of dynamic charts such as water levels, water quality, seepage rates and borehole logs, all within a web browser. The web map interface provides environmental personnel with a quick view of TMF performance via the TARPs (Appendix B).

Data Entry

The web viewer includes an online form that allows data entry for

  • seepage flow rates,
  • groundwater elevation in monitoring wells,
  • surface water and ponding elevations, and
  • settlement data.

Entered data is cross-referenced to the TARPs (as applicable), which assigns a symbology to the station points on the interactive plan map based on the criteria value:

  • green for acceptable readings
  • yellow for an early warning
  • orange when corrective action is required
  • red for critical readings

Data entry and validation request three steps:

  1. Select the station from a drop-down list.
  2. Specify a date via the date picker.
  3. Enter the reading.

Once the reading is entered, several checks occur behind the scenes to compare the incoming reading with the TARPs criteria (as applicable). An appropriate message will be displayed before finalizing the submission by clicking to submit the reading.

Data Quality Data Entry

The water quality chemistry data is loaded via the web viewer using a wizard-based form that walks the user through the import process. The general procedure is as follows:

  1. Select the laboratory file to upload.
  2. Execute a basic QA/QC to determine that the incoming data format is supported.
  3. Click the final import button to load the data to the database.

Any errors or problems during the process will be flagged, and the data will not be allowed to be uploaded until the issues are resolved.

The water quality sampling station names recorded on the chain of command documentation sent to the laboratory must exactly match the nomenclature recorded within the Seabee Mine Environmental Database. Appendix A contains a complete summary of sampling station names.

Temperature Data Entry

Management of the temperature data associated with the thermistor cables in the North Dam seepage collection system is downloaded directly from BeadedStream® at predefined intervals and stored in SRK’s data management system. The data is then entered into the Seabee Mine Environmental Database application for charting and viewing.

Critical Data Reading

If a critical reading has been entered, an automated email will be sent to all environmental personnel, the Site Responsible Person, and the EOR/Deputy EOR, so all applicable persons are aware, and the formulation of an appropriate action plan can begin.

Weather Charts

Specific charts within the Seabee Mine Environmental Database application also incorporate relevant weather data to assess associated influence on, for example, elevation of surface water ponds and flow rate within the North Dam seepage collection system.

Database Outputs

User access to the data will be provided through desktop client options such as Microsoft Excel® or Microsoft Access®. This will enable environmental personnel to utilize the data for further analysis or to generate customized charts.
