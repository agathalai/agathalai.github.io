Projects

The Projects page provides a list of all the projects that are in the system. Projects are used to organize servers and manage session data. Each project can be managed independently, but must be created by a global admin.

The projects list displays the name of each project in alphabetical order.

Click on the far right to show or hide headings that are relevant to you. Select Toggle density to change the amount of spacing in each row.

Add

  1. In the top-right corner, click .

    The My account page is displayed.

  2. From the left menu pane, click Projects.

    If you do not see the left menu pane, swipe to the left browser edge to show the menu.

  3. In the top-right corner of the Projects page, click Add project.
  4. In the following dialog, enter a name for the new project.
  5. Click Add project.

The new project is added to the system. You can now assign users to the project and add servers to it.

Set project permissions

By default, the project uses the global system command permissions defined on the Commands page.

You can choose to enable or disable permissions at the project level, these settings would override the global settings.

To change the project settings, you can access the Project settings page from within a project, or you can access it from the global settings Projects page:

  1. In the top-right corner, click .

    The My account page is displayed.

  2. From the left menu pane, click Projects.

    If you do not see the left menu pane, swipe to the left browser edge to show the menu.

  3. On the Projects page, locate the project you want to modify.
  4. At the right-end of the project row, select > Settings.
  5. On the following Project settings page, make any required changes then click Save.

    For more information about the available settings, see Project settings.

Rename

  1. In the top-right corner, click .

    The My account page is displayed.

  2. From the left menu pane, click Projects.

    If you do not see the left menu pane, swipe to the left browser edge to show the menu.

  3. On the Projects page, locate the project you want to rename.
  4. At the right-end of the project row, select > Rename.
  5. In the following dialog, enter a new name.
  6. Click Rename.

Delete

NOTE: When you delete a project, you will lose access to all connected servers, and session data.

  1. In the top-right corner, click .

    The My account page is displayed.

  2. From the left menu pane, click Projectss.

    If you do not see the left menu pane, swipe to the left browser edge to show the menu.

  3. On the Projects page, locate the project you want to delete.
  4. At the right-end of the project row, select > Delete.
  5. In the following confirmation dialog, click Delete.
  6. If you are prompted, enter your password then click Verify.
