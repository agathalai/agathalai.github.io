Default actions

The Default actions page allows you to set the default actions for triggers.

In the top Default violation actions area, you can set the default response actions for triggers with a risk level of 1-5. Violations are intended to be significant triggers that warrant an alert and are shown on the Home page and Alerts page.

In the Default notice actions area, you can set the default response action for triggers with a risk level of 0. Notices are intended to be low impact triggers that only warrant a system notification.

Actions configured on this page are automatically applied to all triggers of the same type, and are always executed first when the configured trigger conditions are met.

 

To add a default action, click Add action at the bottom of the relevant area. For more information about the available actions, see Trigger actions.

  • You can add as many actions as required.
  • The actions are listed in the order that they will be executed.

    Tip: Make sure you know the order that the actions must be executed because you cannot re-order the actions after they've been added.

  • To delete an action in the sequence, click at the top-right corner of each action.

After you've added or edited all the required actions, click Save at the bottom of the page.
