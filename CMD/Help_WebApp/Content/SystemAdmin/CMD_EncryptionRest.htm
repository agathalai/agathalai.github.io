Encryption at rest

The Encryption at rest page allows you to define when decrypted data is permitted in the system.

Enable or disable any of the following settings, then click Save.

Option Description

Run reports at X:00. Decrypt any extra data needed at this time

Enable this setting to allow data to be decrypted once a day, so that reports can be emailed and viewed outside of the Cmd web interface.

In the following drop down list, select the time when you want the reports to run. The time is listed in 24 hour format.

NOTE: When this setting is enabled, all project administrators automatically receive email notifications of unresolved violation alerts, even when no reports have been created.

If this setting is disabled, you cannot email reports.
Remove decrypted data when all users are logged out Enable this setting to automatically remove decrypted data after all users are logged out of the web interface.
Remove after X hours of inactivity

Enter the number of hours the system must be inactive before decrypted data is automatically removed.
