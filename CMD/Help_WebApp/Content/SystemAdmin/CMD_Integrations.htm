Integrations

The Integrations page lists the third-party tools that integrate with Cmd. Add all the integrations you need to help optimize your daily workflow.

Integrations are project specific so you can assign specific instances of each integration to different projects.

Be aware that the ability to use an integration is enabled globally from the Integration permissions page. For more information, see Integration permissions.

Add

  1. Confirm that you are adding the integration to the correct project.

    The top-right corner of the page displays the project that you are currently accessing.

    If you need to add the integration to a different project, click the name of the current project then select a different project from the drop down list.

  2. In the top-right corner, click .

    The My account page is displayed.

  3. In the left menu pane, click Integrations.

    If you do not see the left menu pane, swipe to the left browser edge to show the menu.

  4. On the Integrations page, click the integration you want to add.
  5. On the selected integration page, click Add integration.
  6. In the following dialog, enter a name for the integration.

    The name helps you identify the integration within Cmd.

  7. Fill in the required fields to connect the integration with Cmd.

    For more information about the required fields, see Integration support.

  8. Click Test integration to verify the connection with the third-party tool.
  9. If you want the integration to be available to all projects in the system, click to enable the Global setting.
  10. Click Save.

Edit

  1. In the top-right corner, click .

    The My account page is displayed.

  2. In the left menu pane, click Integrations.

    If you do not see the left menu pane, swipe to the left browser edge to show the menu.

  3. On the Integrations page, click the type of integration you want to edit.
  4. On the selected integration page, identify the specific integration you want to edit.
  5. At the right-end of the integration row, select > Edit.
  6. In the following dialog, make any required changes.
  7. Click Save.

Delete

  1. On the Integrations page, click the type of integration you want to delete.
  2. On the selected integration page, identify the specific integration you want to delete.
  3. At the right-end of the integration row, select > Delete.
