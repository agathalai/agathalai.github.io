Customizing your Home page

After you log in, the first page you see is the Home page.

If you are a server operator, your Home page will only display your account information. For more information, see Server operator Home page .

For all other users, the top half of the Home page displays charts that show live data from your system. By default, the Home page displays the following charts:

  1. Sessions by location on a map
  2. Session alert frequency
  3. Most alerted commands
  4. First time commands.

    This chart displays a Locked icon because it uses a different base query from the other charts.

  5. Resolved / unresolved violations

The Home page can be customized to show the charts that are most relevant to you.

Tip: If you have access to more than one project, be aware that the charts on the Home page only show data from the selected project. For more information, see Switching projects.

Change chart time range

By default, the charts on the Home page are set to auto refresh every 15 minute and display content from the past month.

To change the chart time settings, select a different option from the related drop down list:

  • Auto refresh — set the auto refresh interval. The options are:
    • Every 15 minutes (default)
    • Every 5 minutes
    • Every minute
    • Every 5 seconds
  • Past <x> — set the time range for the data that is displayed. The options are:
    • Past month (default)
    • Past week
    • Past 24 hrs

NOTE: The selected time period is stored in the browser cookies. If you clear the browser cookies, the chart time settings revert to the system defaults.

Add a chart

By default, all the charts on the Home page use the configured chart time range as the base query:

Where session_date >= '<1 month ago; 1 week ago; 24 hours ago'>

  1. In the top-right corner of the Home page, click Edit.
  2. In the dotted charts area, click to add a chart.

    1. In the following Add Chart dialog, enter a descriptive title for the chart.
    2. In the Query field, add or remove any CQL filters to focus the presented data.

      Click the field to select a new query. Use the auto-complete prompts to complete the query.

      If you remove the blue base query from the field, this icon is displayed over the chart to show that it is unique and not affected by the base query.

      For more information about the CQL filters, see CQL values.

    3. Select one of the available chart types and complete the related options.

      A preview of the chart is displayed with the applied filter.

      For more information about the chart types, see Supported chart types.

    4. Click Save to add the completed chart to the page.
    5. Repeat these steps to add all the charts that are required.
  3. In the top-right corner, click Save to save your changes.

Move or resize a chart

  1. In the top-right corner of the Home page, click Edit.
  2. To move a chart, click and drag the chart to a different position on the Home page.

  3. To resize a chart, click and drag the bottom-right corner of the chart.
  4. Repeat the previous steps until the charts are arranged to your preference.
  5. In the top-right corner, click Save.

Edit or delete a chart

  1. In the top-right corner of the Home page, click Edit.
  2. Hover over a chart then click Edit or Remove.
    • If editing a chart, make any required changes in the following dialog box then click Save. For more information about the available chart options, see Add a chart.
    • If removing a chart, click Yes in the following confirmation dialog
  3. In the top-right corner, click Save.
