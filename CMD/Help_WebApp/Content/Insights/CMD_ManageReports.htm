Edit a report

NOTE: You can only edit reports that you created.

  1. On the Insights page, click the report that you want to edit.
  2. From the top-right corner of the report, click Edit.
  3. Make any required changes then click Done.

For more information about the available report options, see Creating a report.

Email a report

Be aware that after you configure the email settings for a report, the first report is not immediately sent. The first report is sent out the next time the system is configured to run reports. For more information, see Encryption at rest.

For example, today is Tuesday and the system is configured to run reports at 03:00. You set up a report to be sent out every week. The first report is sent out tomorrow at 3 am, then every subsequent Wednesday at 3 am.

  1. On the Insights page, click the report that you want to share.
  2. From the top-right corner of the report, click .

    If the email button is not available, the ability to email reports has been disabled on the Encryption at rest page.

  3. In the following Email report settings dialog, select how frequently the email report is sent.
    • Off
    • Every month (default)
    • Every week
    • Every day
  4. Select all the users who should receive the report. You can use the search bar to find specific users.
  5. Click Save.

Make a report public/private

Reports that are public can be viewed by everyone in the project, but only you can edit the report.

Reports that are private can only be seen or edited by you, but can still be emailed to other users.

  1. On the Insights page, click the report that you want to edit.
  2. From the top-right corner of the report, click Edit.
  3. Click the Private / Public toggle to switch between the visibility modes.
  4. Click Done.

Delete a report

  1. On the Insights page, locate the report you want to delete.
  2. Select > Delete.
  3. When the confirmation dialog is displayed, click Yes.
