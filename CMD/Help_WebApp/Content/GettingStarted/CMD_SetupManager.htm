Manager setup

If you are a manager, you are primarily responsible for managing servers, triggers and reports. Manager permissions are project based, which means you can only make changes to projects you are assigned to.

As a manager, it is recommended that you complete the following steps when configuring the system for the first time:

  1. After you login, it is highly recommended that you setup 2-factor authentication for yourself — 2-factor authentication.
  2. Add servers — see Adding a new server.
  3. Create and manage triggers:
    1. Add triggers — see Adding a trigger
    2. Create macros — see Macros.
    3. Create trigger groups — see Trigger groups.
  4. Create reports — see Creating a report.
  5. Setup your Home page — see Customizing your Home page.

After triggers have been created, alerts will start to come into the system. You will be able to investigate and resolve alerts from the Home page or Alerts page.

NOTE: The help files describe every feature that is available in the system. If you do not see a feature in your version of the web interface, you do not have access to that feature.
