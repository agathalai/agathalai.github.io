Source filters

When you perform a search on the Sources page, you have the option of saving the search query as a filter. The saved filters are only available to your account.

  • Source filters
    • Add
    • Rename
    • Delete

Add

  1. After you perform a search on the Sources page, click then select Sources.
  2. In the following dialog, enter a name for the filter.
  3. Click Yes to save the new filter.

The new filter is added to the left menu pane. If you do not see the left menu pane, swipe to the left browser edge to show the menu.

Next time you want to use the filter, click the name to open the filter in a new tab on the Sources page.

Rename

  1. On the Sources page, locate the filter that you want to rename from the left menu pane.

    If you do not see the left menu pane, swipe to the left browser edge to show the menu.

  2. Click beside the filter.
  3. In the following dialog, change the filter name.
  4. Click Save.

Delete

  1. On the Sources page, locate the filter that you want to delete from the left menu pane.

    If you do not see the left menu pane, swipe to the left browser edge to show the menu.

  2. Click beside the filter.
  3. In the following dialog, click Delete.
