Supported chart types

This page provides a list of the supported system chart types, and the settings that are specific to each chart.

Charts are used on the Home page and in reports to help visualize the data collected by the system.

  • Supported chart types
    • Numbers
    • Text
    • List
    • Line
    • Bar
    • Pie
    • Map
    • Calendar

Numbers

This type of chart simply displays the total count of the selected property. You can modify the following:

  • Value — select any one of the listed values that can be totaled and displayed.
  • Color — select from any of the available color options.

Text

This type of chart simply displays text that you enter in the Text field.

List

This type of chart provides a list of total values within the selected parameters. You can modify any of the following:

  • Parameter — select any one of the listed parameters to populate the list items.
  • Value — select any one of the listed values to display it in relation with the selected parameter.
  • Color — select from any of the available color options.
  • Sort — this option appears for parameters that have a larger dataset than others. You can choose to limit the listed data by number of segments and sort the data in ascending or descending order.

Line

This type of chart shows the selected dataset as a line graph. You can modify the following:

  • X-axis — select any one of the listed parameters to populate the x-axis.
  • Y-axis — select any one of the listed values that can be totaled and displayed according to the selected x-axis.
  • Color — select from any of the available color options.
  • Sort — this option appears for parameters that have a larger dataset than others. You can choose to limit the listed data by number of segments and sort the data in ascending or descending order.
  • Show summary — enable this setting to show a total count of all the y-axis values.

Bar

This type of charts shows the selected dataset as a bar graph. You can modify the following:

  • Layout — choose if the chart would display the bars in a horizontal or vertical layout.
  • X-axis / Y-axis — select any one of the listed options to populate the chart. The values change depending on the selected layout.
    • If you are adding a time based chart, click to display the Violation count values for comparisons. By default, the data comparison is shown side by side. If you prefer to show the data on top of each other, enable the Stacked option.

      Click to remove the Violation count comparison.

  • Color — select from any of the available color options.
  • Sort — this option appears for parameters that have a larger dataset than others. You can choose to limit the listed data by number of segments and sort the data in ascending or descending order.
  • Show summary — enable this setting to show a total count of all the horizontal-axis values.

Pie

This type of chart provides a list of values for the selected parameters, and displays the same data in a pie chart. You can modify the following:

  • Parameter — select any one of the listed parameters to populate the chart.
  • Value — select any one of the listed values to display it in relation with the selected parameter.
  • Color — select from any of the available color options.
  • Sort — this option appears for parameters that have a larger dataset than others. You can choose to limit the listed data by number of segments and sort the data in ascending or descending order.
  • In the top-right of the chart preview area, select to display the pie chart as a Half or Full circle.

Map

This type of chart displays the selected values based on origin. You can modify the following:

  • Listing — select the preferred map level for labeling the map values:

    In the top-right corner of the map preview, click to see an example of the selected listing type.

    • none — The icon is disabled.
    • Countries
    • Regions — province or state
    • Cities.
  • Value — select any one of the listed values to display its origins on the map.
  • Color — select from any of the available color options.

Calendar

This type of chart displays the selected values based on the calendar range. You can set the time range by using the Query filter. If no time range is entered, the chart defaults to the past month.

You can modify the following:

  • Value — select any one of the listed values to display it on the calendar.
  • Color — select from any of the available color options.

    • Tip: Do not select the red color option if you are also enabling the Show violations setting because violations are automatically displayed in red.

  • Show violations — enable this setting to highlight any violations that are detected within the same time range.
