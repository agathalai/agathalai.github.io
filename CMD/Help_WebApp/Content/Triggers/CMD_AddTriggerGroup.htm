Trigger groups

To help you organize the triggers in the system, you can create trigger groups. If there are no configured trigger groups in the system, all triggers are automatically added to the Generic group.

From the left menu pane of the Triggers page, you can click a trigger group to filter the listed triggers down to the ones assigned to the selected group.

  • Trigger groups
    • Add
    • Remove

Add

  1. On the Triggers page, locate Trigger groups in the left menu pane.

    If you do not see the left menu pane, swipe to the left browser edge to show the menu.

  2. Beside Trigger groups, click .
  3. In the following dialog, enter a name for the new group.

    Tip: Check your spelling because you cannot edit trigger group names.

  4. Select all the triggers that should be part of the group.

    Use the search function to find any triggers that you do not immediately see.

  5. Click Add trigger group.

Remove

You cannot delete a trigger group unless there are no triggers assigned to it.

  1. In the left menu pane, click the trigger group that you want to delete.
  2. Remove any triggers that may be listed.

    To remove triggers from the group, edit each listed trigger and assign it to a different group.
