System Setup Checklist

Installer:

Project Name:

Install and configure the ACC system as follows:

Important: Always follow system design documentation and criteria for all device and server settings.

  1. Install cameras and devices.

    For more information, see Install Hardware and Software.

    1. Connect devices to network.
    2. Aim and focus cameras.
    3. Assign a name and location for the camera or device.
    4. Assign a dynamic or static IP address to the camera or device.
  2. Install the video recorder.
    • Windows based NVR — NVR3 or HD Video Appliance
      1. Complete initial Windows setup.
      2. Set date and time.
      3. Set computer name.
      4. Set new password for local administrator account.
      5. Activate site license according to system design. See Activate Site Licenses.
    • ACC ES Recorder or ACC ES Analytics Appliance
      1. Assign password to administrator account in the web interface.
      2. Assign a hostname for the recorder.
      3. Set date and time.
      4. Set a name for the recorder.
  3. Configure NTP time synchronization.
  4. Install and run ACC Client software on local workstation.
  5. Configure sites and servers:
    1. (Enterprise systems only) Merge multiple servers into a single site as required. See Connecting Servers to Sites.
      • Activate licenses for the new site. See Activating a License for the First Time.
    2. Configure the Site View. See Editing the Site View.
    3. Connect cameras to the servers. See Connecting a Device to a Server.
    4. Enable analytics devices. See Configure Video Analytics.
      • Enable self-learning. See Enabling Self-Learning.
  6. Configure devices:
    1. Assign a logical ID to the camera. See Setting a Device's Identity.
    2. Adjust video image and display. See Changing Image and Display Settings.
    3. Set compression and image rate. See Compression and Image Rate
      • Image rate.
      • Quality level.
      • Keyframe interval.
    4. Configure motion detection areas.
      • Pixel Motion. See Setting Up Pixel Motion Detection.
        • Green motion detection area.
        • Sensitivity.
        • Threshold.
      • Classified Object Motion. See Setting Up Classified Object Motion Detection.
        • Green motion detection area.
        • Type.
        • Sensitivity.
        • Threshold.
    5. Recording schedule. See Recording Schedule.
    6. Data aging settings. See Recording and Bandwidth.
  7. Add users and groups. See Add Users and Groups.
  8. Configure Avigilon Rules and Alarms as required to satisfy all system functionality per the system design documentation.
  9. Customize video monitoring setup
    • Add saved Views. See Saved Views.
    • Add maps. See Adding a Map.
    • Configure joysticks. See Joystick Settings.
  10. Configure external notifications. See External Notifications.
  11. Configure ACC Mobile access.
    • To run ACC Mobile 2.x, install and configure ACC Gateway software.
    • To run ACC Mobile 3.x, install ACC Web Endpoint software.
  12. Verify setup — Log in as different users to check interface and permissions.
  13. Download ACC mobile from the Apple store or Google Play store.
    • For ACC Mobile 2.x, configure the app to point to a Gateway IP address.
    • For ACC Mobile 3.x, configure site address to point to an ACC IP address.
