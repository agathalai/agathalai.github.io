Editing the Search Parameters

Search parameters can be refined once some search results have been populated. Refining search parameters allows you to adjust the time range, duration of the search, cameras included in the search, confidence level and order of the search results. Search parameters can be refined at anytime during the search.

For example, if the man is not seen arriving in the red truck at the beginning of the time range, the time range can be extended.

  1. In the top-left corner of the Search: Appearance tab, click Refine Search.
  2. In the Refine Search dialog, adjust the search parameters.
    • To include specific cameras in the search, select the check box beside the camera. To select all the cameras, select the check box beside the site. Only cameras with the feature enabled are displayed.
    • In the Date Range options, adjust the From: and To: fields, or adjust the Duration: field.

      The maximum duration of the search is 1 day. The duration of the search can also be adjusted by dragging the black time range markers on the Timeline.

  3. Click Search.

    The search results update with the new parameters.
